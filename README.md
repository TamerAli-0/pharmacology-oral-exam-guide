# Pharmacology-2 — Oral Exam Study Guide

High-yield, spoken-answer revision covering all **62 examinable topics** for the
Pharmacology-2 oral examination, aligned to *Lippincott Illustrated Reviews: Pharmacology*.

**📖 Read it online:** https://tamerali-0.github.io/pharmacology-oral-exam-guide/

Every topic follows the same oral-exam structure: Definition → Mechanism of action →
Classification → Pharmacokinetics → Therapeutic uses → Adverse effects →
Contraindications → Oral exam pearls → a recitable **30–60 second answer**.
